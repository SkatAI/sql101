# Course Curriculum

- **SQL Query**: the essential skill
  - filtering, projections, joins, keys, constraints
- **database design**: construct efficient schemas, apply normalization when appropriate
  - normalization : when to keep data unique or allow duplicates
- **data processing**: transform the data, constraints
  - functions, data quality, null values
- **Index Creation and Management**:
  - Understand the importance of **indexes** in improving database performance
  - learn to create and manage different types of indexes (B-tree, hash, etc.).
  - execution plan analysis

---

# things we won't cover

- **Views and Stored Functions**:
  - create views to simplify complex queries
  - use stored functions to **encapsulate business logic** within the database - **PLSQL**

- **Transactions and Concurrency Control**: ensure data integrity in multi-user environments.

- **Database Security**:  roles and permissions,

- **Maintenance and Monitoring**: Preventive maintenance techniques and monitoring to ensure database availability and performance.
- **Advanced querying**: CTEs, Window functions
- **Cloud**: Bigquery

# Course Curriculum

By the end of this course, you will be able to:

1. **Design** optimized relational databases for various applications.
2. Write efficient and **high-performance** SQL queries.
